# 나만의 프롬프트 관리 프로그램

## 프로젝트 소개

Python으로 만든 콘솔 기반 프롬프트 관리 프로그램입니다.

사용자가 AI 프롬프트를 등록하고, 목록 조회, 카테고리 조회, 검색, 상세 보기, 즐겨찾기 기능을 통해 프롬프트를 효율적으로 관리할 수 있습니다.

프롬프트 추가,

## 실행 방법

1. 저장소를 다운로드하거나 클론합니다.
2. 프로젝트 폴더로 이동합니다.
3. 아래 명령어를 실행합니다.

```bash
python main.py
```

---

## 주요 기능

- **프롬프트 추가** : 새로운 프롬프트를 등록합니다.
- **프롬프트 목록 조회** : 등록된 모든 프롬프트를 확인합니다.
- **카테고리별 조회** : 선택한 카테고리의 프롬프트만 조회합니다.
- **프롬프트 검색** : 제목 또는 내용에 검색어가 포함된 프롬프트를 찾습니다.
- **프롬프트 상세 보기** : 선택한 프롬프트의 제목, 내용, 카테고리, 즐겨찾기 상태와 조회수를 확인합니다.
- **즐겨찾기 등록 및 해제** : 프롬프트의 즐겨찾기 상태를 변경합니다.
- **즐겨찾기 목록 조회** : 즐겨찾기로 등록한 프롬프트만 확인합니다.
- **프롬프트 수정** : 등록된 프롬프트의 제목, 내용, 카테고리를 변경합니다. (보너스2)
- **프롬프트 삭제** : 선택한 프롬프트를 목록에서 삭제합니다. (보너스2)
- **조회수 기록** : 프롬프트 상세 보기 시 조회수를 1씩 증가시킵니다. (보너스2)
- **조회수 상위 목록** : 조회수가 높은 프롬프트를 내림차순으로 정렬해 보여줍니다. (보너스2)
- **프롬프트 저장 및 불러오기** : 프롬프트 데이터를 JSON 파일에 저장하고 프로그램 시작 시 다시 불러옵니다. (보너스1)
- **Markdown 내보내기** : 전체 프롬프트를 카테고리별 Markdown 파일로 내보냅니다. (보너스1)
- **프로그램 종료** : 프로그램을 종료합니다.
